# Titanic Data Cleaning Project

This project demonstrates basic data cleaning using Python and pandas.

## Dataset
The Titanic passenger dataset containing information about passengers such as age, ticket class, fare and survival.

## Steps Performed

1. Loaded the dataset using pandas
2. Inspected the dataset structure using df.info()
3. Checked for missing values using df.isnull().sum()
4. Removed rows with missing values using dropna()
5. Saved the cleaned dataset as titanic_cleaned.csv

## Tools Used
Python  
Pandas  
Jupyter Notebook

## Goal
Practice basic data cleaning techniques used in data science workflows.
